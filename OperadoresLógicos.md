# Operadores Logicos 


| Operador      |  Definicion   |
| ------------- |:-------------:| 
| &&            | and           |
|     (|)       | (or)          |
| XOR           | Or Exclusivo  |

##  Operadores de relacion

| Operador      |  Definicion   |
| ------------- |:-------------:| 
| >             | Mayor que     |
| <             | Menor que     |
| >=            | Mayor o igual a  |
| <=            | Menor o igual a  |
| ==            | Igual a       |
| ~=            | Diferente de |

## Instruccion _if-end_

   Si la ejecucion se cumple, se ejecutan las declaraciones hasta el *end* 
   si no lo cumple se salta el ciclo 
   
   
    if (condicion)
      Declaraciones 
        end;
